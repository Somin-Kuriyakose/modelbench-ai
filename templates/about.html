About the Project

Built for engineers
who care about performance

ModelBench AI is a self-hosted benchmarking platform that gives machine learning practitioners deep, honest insight into how their models behave under real inference conditions — latency, memory, CPU, and accuracy.

"You can't optimize what you can't measure — and most ML teams are flying blind on inference performance." — Core Design Philosophy

ModelBench AI was built to solve a real problem: shipping models to production without a clear picture of their latency tail, memory footprint, or how batch size would affect throughput. Existing tools were either too cloud-centric, too opaque, or locked to a single framework.

ModelBench is clean, local, and framework-agnostic. It produces reproducible, detailed results you can actually act on — no telemetry, no subscriptions, no data leaving your environment.

The platform supports async job execution, real-time memory profiling, CPU utilization tracking, accuracy metric scoring with label upload, automatic model introspection, and batch-size sweep — all from a single drag-and-drop interface.

Capabilities

Every metric that matters, in one place

⚡

Latency Profiling

Captures avg, median, p50/p90/p95/p99, min, max and std deviation across hundreds of inference iterations with configurable warmup runs.

🧠

Memory & CPU Tracking

Tracks peak heap allocation via tracemalloc, process RAM via psutil, and CPU utilisation sampled per inference run — not just a snapshot.

🎯

Accuracy Metrics

Upload ground-truth labels to automatically compute accuracy and F1 for classifiers, or RMSE, MAE and R² for regression models.

🔬

Model Introspection

Surfaces class name, parameter counts, n_features, n_classes, feature importances, and model size without any manual inspection required.

📊

Batch Size Sweep

Automatically profiles your model across multiple batch sizes in one shot and identifies the optimal configuration for latency vs throughput.

🔄

Compare & Export

Tag and annotate results, compare benchmarks side-by-side with best/worst highlighting, and export any result or all results as CSV.

Our Principles

What ModelBench stands for

Privacy First

Your model weights and data never leave your machine. ModelBench runs entirely locally — no outbound telemetry, no cloud APIs, no user tracking of any kind.

Framework Agnostic

Works with sklearn, PyTorch, TensorFlow, and ONNX Runtime through a single unified interface. Switching ML frameworks should never mean switching tools.

Honest Metrics

We report p99 tail latencies, not just averages. Peak memory, not current usage. The numbers that matter in production are always shown — never hidden.

Zero Lock-in

Export everything to CSV. Results are plain JSON on disk. The platform is fully open. You own your workflow and your data, completely and always.
