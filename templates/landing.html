How it works

Four steps to insight

01

Upload Model

Drop in your .pkl, .pt, .h5 or .onnx model file. Pickle security warnings shown automatically.

02

Upload Data

Provide test data as .csv, .npy, or .npz. Optionally upload a labels file for accuracy scoring.

03

Configure & Run

Set batch size, iterations, warmup runs. Run synchronously or dispatch as an async background job.

04

Analyse & Export

Explore interactive charts, compare runs side-by-side, tag results, and export to CSV.
